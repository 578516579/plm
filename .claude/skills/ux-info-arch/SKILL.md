# ux-info-arch — UX 信息架构 skill v0.1

## 1. 何时调用

## 2. 维度

### 2.1 菜单层级
- 一级菜单 ≤ 7 (per Miller's law)
- 嵌套 ≤ 3 层

### 2.2 页面信息密度
- 列表页: 列数 ≤ 8 (移动端 ≤ 4)
- 表单: 字段分组, 每组 ≤ 5 字段

### 2.3 字段优先级
- 主信息 (一目了然)
- 次信息 (折叠 / hover 展示)
- 三级信息 (详情页才显)
