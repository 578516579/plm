## 1. 背景与目标

### 1.1 现状痛点

研发团队当前用飞书多维表格 + 散落的飞书文档管理项目。带来 3 个具体问题：

1. **状态不一致**：飞书表格、客户邮件、Sprint 看板上的同一项目状态经常对不上，每次复盘要花 30 分钟"对账"。
2. **追溯困难**：项目立项时的预算、阶段计划、风险登记散在不同文档，2 个月后想看"为什么当时这样定"要翻 3-5 处。
3. **数据割裂**：项目 → 阶段 → 任务 → 需求 没有强外键，统计"某项目下所有任务进度"靠人肉聚合。

### 1.2 目标（北极星指标）

**目标**：12 个月内沉淀 ≥ 50 个项目的完整生命周期数据，作为后续阶段管理 / 任务管理 / 需求管理等子实体的承接载体。

**衡量指标**：
- 月活项目数 ≥ 30（活跃定义：当月有状态变化或新增评论）
- 数据完整率 ≥ 90%（含必填字段：项目编号 / 名称 / 类型 / 状态 / 负责人 / 起止日期 / 预算）
- 跨表关联率 ≥ 70%（每个 active 项目下有 ≥ 1 个关联子实体记录）

### 1.3 不做的事（Out of Scope）

本期**不做**：
- 项目模板复制（"基于已有项目快速建新项目"）— Phase 2 再做
- 项目甘特图可视化 — 等阶段/任务模块就位后再做
- 项目级权限隔离（"A 项目的成员看不到 B 项目"）— v0.2 做，本期所有 admin 可见
- 项目门户 / 对外公开页 — 不在 PLM 系统范围内
- 工时统计 / 财务对账 — 与现有飞书系统对接，本期不重做

## 2. 用户与场景

### 2.1 角色

| 角色 | 在系统中的权限 | 典型动作 |
|---|---|---|
| **项目经理（PM）** | CRUD 自己负责的项目 | 立项、状态推进、关联子实体 |
| **管理员（admin）** | 全 CRUD | 查看全部、调整状态、删除 |
| **普通成员** | 查看 + 评论 | 跟进进度、留言反馈 |

> 普通成员的"评论"功能 v0.2 做，本期只先开 CRUD。

### 2.2 典型场景

**S1 立项**（最高频）
> 王经理想新立一个"客户 X 定制开发"项目 → 进入项目管理 → 新增 → 填编号 / 名称 / 类型=研发 / 负责人=自己 / 起止 / 预算 → 状态默认"未启动" → 保存 → 在列表中看到。

**S2 状态推进**
> 项目实际启动 → 王经理改状态为"进行中" → 系统记录修改人 + 时间。

**S3 看板筛选**
> 总监想看本月所有"进行中"的研发类项目 → 列表 → 类型=研发 + 状态=进行中 + 起止日期落在本月 → 一屏看完。

**S4 导出汇报**
> 周会前 → 选中本周变化的项目 → 导出 Excel → 贴飞书周报。

### 3.3 状态机

```
未启动(0) ──启动──▶ 进行中(1) ──暂停──▶ 暂停(2) ──恢复──▶ 进行中(1)
                       │                                          │
                       └─完成──▶ 已完成(3)         任意非"已完成"──▶ 已取消(4)
```

合法转换矩阵：

| 当前 → | 0 未启动 | 1 进行中 | 2 暂停 | 3 已完成 | 4 已取消 |
|---|---|---|---|---|---|
| 0 未启动 | — | ✅ | ❌ | ❌ | ✅ |
| 1 进行中 | ❌ | — | ✅ | ✅ | ✅ |
| 2 暂停 | ❌ | ✅ | — | ❌ | ✅ |
| 3 已完成 | ❌ | ❌ | ❌ | — | ❌ |
| 4 已取消 | ❌ | ❌ | ❌ | ❌ | — |

非法转换由 Service 层拦截，抛 ServiceException(701)。

## 4. 非功能需求

| 维度 | 要求 |
|---|---|
| **性能** | 列表查询 P95 ≤ 500ms（≤ 1 万项目）；导出 1000 行 ≤ 3 秒 |
| **可用性** | 走若依现有部署，跟系统其他模块同 SLA |
| **安全** | 权限走 `business:project:list/query/add/edit/remove/export`；导出无 path 注入风险 |
| **审计** | 写操作全部 `@Log` 注解 |
| **数据完整性** | `del_flag` 逻辑删除；不允许物理 DELETE |
| **兼容性** | 浏览器支持 Chrome / Edge / Safari 最近 2 个版本 |
