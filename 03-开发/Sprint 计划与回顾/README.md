# Sprint 计划与回顾

每个 Sprint 一个独立 .md，含计划、看板状态、每日站会要点、回顾纪要。

## 命名规范

`Sprint-NN-YYYY-MM-DD_to_YYYY-MM-DD.md`，例如 `Sprint-01-2026-05-15_to_2026-05-29.md`

## 模板

```markdown
# Sprint NN（YYYY-MM-DD ~ YYYY-MM-DD）

## 目标（Sprint Goal）
> 本期最重要要达成什么。一句话。

## 计划纳入
| 任务 | 估时 | 负责人 | 优先级 |
|---|---|---|---|

## 每日要点
- YYYY-MM-DD：xxx
- ...

## 回顾（Retrospective）
- **做得好**：
- **可改进**：
- **行动项**（带 DDL）：
  - [ ] xxx

## 完成情况
| 计划工时 | 实际工时 | 完成率 |
```

## 看板工具

建议同步到 Jira / 禅道 / GitHub Projects；本目录只存 Markdown 化的快照与回顾。
