## 1. 测试范围

### In Scope（本期测）

- `ProjectServiceImpl` 业务逻辑（含 generateProjectNo / 状态机 / 字段校验）
- `ProjectController` 路由 + 权限拦截 + 错误码（通过 Phase 03 E2E 等价验证）
- `ProjectMapper` 与 DB 交互（通过 E2E 验证）
- 字典与菜单数据完整性

### Out of Scope（不测）

- 性能（无明确性能 SLA；Project 量级 < 万，MySQL+索引覆盖足够）
- 并发压测（[需要时另起 ADR](../03-开发/ADR/0001-project-no-rule.md) 跑 `generateProjectNo` 并发用例）
- UI 自动化（无独立 UI 组件，沿用 Element Plus 标准）
- 跨浏览器兼容性（沿用若依默认支持范围）
