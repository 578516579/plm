## 5. 风险

| 等级 | 风险 | 缓解 |
|---|---|---|
| P0 | 多人同时编辑同一 Sprint 看板 / 卡片导致乐观锁冲突或脏写 | update_time 乐观锁 + 服务端 version 字段 + 前端冲突提示返回 409 |
| P0 | 看板拖拽顺序错乱 (前端 order 字段未持久化或并发拖拽冲突) | 服务端排序字段权威 + 拖拽后 PUT /reorder 立即落库 + E2E 覆盖并发拖拽 |
| P1 | 状态机非法转换 (跳过中间状态或反向转换) | Service 单测覆盖合法+非法矩阵 + transitTo() 白名单 + 返 601 |
| P1 | FK 级联删除产生孤儿数据 (如删除 Sprint 时关联 Task 未处理) | E2E 覆盖删除场景 + DBA review FK + 软删 del_flag 标记代替硬删 |
| P2 | 高频列表分页查询性能 (任务量 N≥1000 时拖慢看板加载) | EXPLAIN 关键查询 + 关键列建索引 + 数据量 1000+ 压测 |
