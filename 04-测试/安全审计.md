# 安全审计

记录每次安全扫描、渗透测试、依赖漏洞核查的结果与处置。

## 1. 审计周期
> 每个 release 前 / 每月 / 每季度——本项目选哪种？

## 2. 检查项 Checklist
- [ ] **依赖漏洞**：`mvn dependency:tree` + OWASP Dependency-Check / `npm audit`
- [ ] **OWASP Top 10**：注入、鉴权、XSS、CSRF、SSRF、文件上传、反序列化
- [ ] **凭据外泄**：git history grep、`.env` 是否被提交、JWT secret 是否仍是默认弱值
- [ ] **接口越权**：水平/垂直越权用例
- [ ] **日志脱敏**：手机号、身份证、密码字段是否打码
- [ ] **HTTPS / HSTS**：生产入口
- [ ] **Druid 监控台**：是否对外暴露、是否仍用默认口令
