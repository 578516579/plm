# 测试用例库

按业务模块组织的测试用例集合。优先用 Markdown 表格便于评审，重型用例用 Excel/飞书多维表格导出后存在这里。

## 文件命名

- `<模块>-functional.md`：功能用例
- `<模块>-api.md`：接口用例（含 method/path/请求/期望）
- `<模块>-e2e.md`：端到端场景

## 用例模板（功能）

| 用例 ID | 标题 | 前置条件 | 步骤 | 预期 | 优先级 | 状态 |
|---|---|---|---|---|---|---|
| TC-001 | xxx | xxx | xxx | xxx | P0 | 待执行 / 通过 / 失败 |

## 与代码的关系

- 单元测试代码：留在源码 `src/test/` 下，不入本目录
- 自动化脚本：放在 `04-测试/` 下另开 `automation/` 子目录，或独立仓库引用
