# 06-运营 — Phase 06 Operations

> Phase 06 是模块上线后的**持续优化阶段**,采集用户反馈、运营指标、缺陷数据,反哺到下一轮 Phase 01 立项。详 [99-跨阶段/模块工作流.md Phase 06](../99-跨阶段/模块工作流.md)。

## 当前模块清单

Phase 06 不强制每模块产出,**只为重要模块或问题模块写运营快报**。当前 Project 模块作 v0.1.0 首发,已走完 cycle 1。
