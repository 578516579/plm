# 周报 / 月报

固定节奏的运营汇报，便于历史回顾。

## 命名规范

- 周报：`YYYY-WW-周报.md`（W 为 ISO 周序号），例如 `2026-W20-周报.md`
- 月报：`YYYY-MM-月报.md`，例如 `2026-05-月报.md`

## 周报模板

```markdown
# YYYY 第 WW 周周报（YYYY-MM-DD ~ YYYY-MM-DD）

## 核心指标
| 指标 | 本周 | 上周 | 环比 |

## 完成的事
-

## 进行中
-

## 下周计划
-

## 风险 & 求助
-
```

## 月报建议

- 含本月核心指标 + 趋势图（折线）
- 用户反馈摘要（见 [../用户反馈/](../用户反馈/)）
- 上线版本汇总（指向 [05-上线/Changelog.md](../../05-上线/Changelog.md)）
