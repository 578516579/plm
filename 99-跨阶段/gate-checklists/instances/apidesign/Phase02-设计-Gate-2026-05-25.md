**关键路径时序**

```
路径1 — 新建 + AI 生 OpenAPI
用户填表 (项目+method+path+title) → Vue formRef.validate() → POST /business/apidesign
       → JwtAuthenticationTokenFilter
       → ApiDesignController.add (PreAuthorize business:apidesign:add)
       → ApiDesignServiceImpl.insertApiDesign
       → 字段校验 (title/projectId/httpMethod/path/authorUserId)
       → projectMapper.selectProjectById 存在性 (不存在 → 702)
       → ENUM 白名单 (httpMethod ∈ {GET,POST,PUT,DELETE,PATCH,HEAD,OPTIONS}, 否则 604)
       → 默认值 (mockEnabled=N / aiGenerated=N / status='00')
       → 新建非草稿 → 601
       → generateApiDesignNo() → APID-YYYY-NNNN
       → mapper.insertApiDesign
         catch DuplicateKeyException:
           - uk_apidesign_project_method_path → 抛 701 "项目内 method+path 已存在"
           - uk_apidesign_no → 重号重试 1 次
       → 返回 apidesignId

AI 生成 → 用户在详情卡点 ✨ AI 生成接口设计 → POST /business/apidesign/ai/generate/{id}
       → ApiDesignServiceImpl.aiGenerate
       → aiService.chat(AiChatRequest 含 system+user+callerTag) [审计落表]
       → mock 输出 standard OpenAPI 3.0.3 YAML 片段
       → 回填 openapiSpec / requestSchema / responseSchema / mockResponse
       → aiGenerated='Y' + aiGeneratedAt=now
       → mapper.updateApiDesign
       → 返回 ApiDesign 全字段

路径2 — 评审 lifecycle(含反向边)
PUT { status: '01' }    → 草稿 → 评审中
PUT { status: '00' }    → 评审中 → 草稿 (反向打回 ✅ unique to apidesign)
PUT { status: '02' }    → 评审中 → 已确认
PUT { status: '03' }    → 已确认 → 已废弃 (终态)
非法转换 (e.g. 00→02 跳级 / 02→01 / 03→任意) → 抛 601 with statusLabel
```

**部署拓扑**:dev/staging/prod 单实例;AI 走 MockAiProvider 兜底,不阻塞业务。

**集成点**:sys_user(author/reviewer FK)/ sys_dict_data(biz_apidesign_status)/ sys_menu / sys_oper_log / JWT 链 / **AiService 审计表 ai_invocation_log** / **plm-project 项目存在性校验** / **plm-arch 架构 FK(可选,本期 UI 未暴露 archId 输入)**。

- [x] 权限串符合 `business:<entity>:<action>` 规范
- [x] **错误码**(已登记 [PRD-MAPPING.md §4](../../../../PRD-MAPPING.md)):
  - **404** `接口设计不存在` — update/aiGenerate 时主键不存在
  - **601** `状态机非法转换(草稿 → 评审中 ⇄ 草稿 → 已确认 → 已废弃)` / `新建接口设计状态必须为「草稿」`
  - **602** `必填字段缺失(接口设计标题 / 关联项目 / HTTP 方法 / 接口路径 / 设计者)`
  - **604** `HTTP 方法仅支持 GET/POST/PUT/DELETE/PATCH/HEAD/OPTIONS`
  - **701** `项目内已存在相同 method+path 的接口` — `uk_apidesign_project_method_path` 唯一键冲突
  - **702** `关联项目不存在`
- [x] 鉴权方式:JWT Bearer + `@PreAuthorize`
