# 会议纪要

跨阶段会议（如周会、Review、对齐会）的纪要存档。立项评审纪要单独放在 [01-立项/立项评审纪要.md](../../01-立项/立项评审纪要.md)。

## 命名规范

`YYYY-MM-DD_<会议类型>_<主题>.md`，例如：
- `2026-05-15_周会_本周进度同步.md`
- `2026-05-20_对齐会_API 设计.md`

## 模板

```markdown
# 会议纪要：<主题>

- 日期：YYYY-MM-DD HH:MM
- 主持：
- 记录：
- 参会：

## 议程
1.
2.

## 关键讨论与决议
> 议题 → 结论 → 责任人 → 截止日。

## 行动项
- [ ] xxx — 负责人 — DDL
- [ ] xxx — 负责人 — DDL

## 下次议题
-
```
