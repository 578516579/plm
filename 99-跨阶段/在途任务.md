## 状态语义

| 状态 | 含义 |
|---|---|
| 进行中 | 当前 session 活跃，正在写代码 / 文档 |
| 等评审 | PR 已开，等 review |
| 等合并 | review 已 approve，等 SSoT 串行槽位 / CI 通过 |
| 已合并 | 进入 main，挪到"已完成"段 |
| 暂停 | 因依赖 / 阻塞 / 转交其他活，搬到"暂停 / 等待"段 |
| 放弃 | 不再继续，挪到"已完成"段并在备注里写原因 |

---

## 使用约定

- **开工**：在"进行中"加一行（按时间倒序，新行加在最上）
- **状态变更**：原地改"状态"列
- **合入 main**：从"进行中"挪到"已完成"（保留 7 天后可删）
- **超过 24 小时未更新的"进行中"行**：可由任何 session 提问"是否还在跑"，无答复则挪到"暂停"
- **冲突预防**：开工前看是否有人在改你要改的模块/范围；如重叠先 ping 该行 Owner
