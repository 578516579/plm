## 字段说明

- **类型**：技术 / 业务 / 人力 / 合规 / 外部依赖 / 安全
- **概率**：低 / 中 / 高
- **影响**：低 / 中 / 高
- **等级**：P0（必须立刻处理）/ P1（本期处理）/ P2（持续关注）
- **状态**：open / mitigating / closed / accepted

## 评审节奏

- 每周 Sprint 计划时复审一次（见 [03-开发/Sprint 计划与回顾](../03-开发/Sprint%20计划与回顾/)）

- 重大变更（架构 / 发布）前强制复审
